## 📦 Installation (Internal)

1. Mount the `Wizard-v3.0.0.dmg` (Production Build 3.0.0).
2. Drag **Wizard Suite.app** to your `/Applications` folder.
3. On first launch, if prompted, allow permissions for the app to access your Work/Test folders.
